The **Inputs/Outputs** menu configures how the display receives and sends data — CAN bus messages, serial ports, analog and frequency inputs, cameras, and more.

Each input writes to a channel. Each output reads from a channel or triggers an event. See [Channels and Events](../concepts/channels-and-events) for how data flows between inputs, functions, and outputs.
