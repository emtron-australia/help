## Display Studio Release Notes 1.0.2

### New Features

 - Bluetooth Audio (ED10/7 can act as a Bluetooth head unit).
 - CANOpen supports signed numbers
 - Updated user interface font
 - Ability to import track start/finish and sectors from a track geoJSON file

### Bug Fixes

 - Fixed copy and paste not working in the layer editing screen
 - Fixed analog channel losing assignment in ED7
 - Fixed OBDII screen layout issue

## Display Studio Release Notes 1.0.1

### New Features

 - Fully overhauled the logging system to use an ultra efficient storage format with no limit on log size
 - Added a web interface for downloading logs
 - Added multilingual support: (English, Japanese, Spanish, Chinese, French, German, Portuguese, Italian, Russian, Korean)
 - Added IMU orientation calibration feature
 - Added additional lap and sector statistics and deltas
 - Adding adjustable overlay font sizes
 - Adding default colors to status gauges
 - Added support for connecting to a telemetry server
 - Added the ability to decode floating point values from CAN messages

### Bug Fixes

 - Fixed issue where the application wouldn't reconnect after device restart
 - Fixed various crashes when using the application
