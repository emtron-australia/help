**Engine Main**


Engine Main allows for configuring the ECU to suit the engine. &nbsp;

The following settings are configured in this menu item :


Cylinders&nbsp;

Engine Size

Engine Type

Crank RPM Exit&nbsp;

Crank RPM Entry&nbsp;

Engine Speed Valid

Engine Speed Invalid
