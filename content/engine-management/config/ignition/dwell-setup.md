## Dwell Setup

![Image](</img/Ignition10.jpg>)

## Overview

There are two Dwell Tables available. This setting determines which Dwell Table is assigned to each  cylinder.

In most applications all cylinders will use the same Dwell Table. However, multiple dwell tables can be useful when different ignition coil types are used, or when leading and trailing ignition systems require different dwell characteristics.
