## Requirements
The following are the minimum requirements to be able to utilise Emtune software

 - Laptop or Desktop
 - Windows 7/8/10/11 (32 or 64 bit) in English as native language
 - Ethernet Port (Recommended) or USB to Ethernet adapter
 - Internet Connection
