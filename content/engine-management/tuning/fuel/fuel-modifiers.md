---
title: "Fuel Modifier Tables"
---

Fuel Modifier tables can be turned on/off.  

These tables compensate the raw air mass or fuel mass calculation within the model

![Image](</img/NewItem742.png>)

**OFF –** The Compensation Table is Off

**ON -**  The Compensation Table is On 

Each compensation table will appear individually under Fuel -> Fuel Modifier Tables

![Image](</img/NewItem743.png>)

## Fuel Mass Modifier Table"
Fuel Mass Modifier Table is enabled via : Tuning -> Fuel Table Control -> Fuel Modifier Tables

![Image](</img/NewItem355.png>)

Values in this table modify the ECUs **Final** Fuel Mass directly as a percentage
