## Engine Type

This entry designates the engine type. There are 2 type of engine which the Emtron ECU's can control :

**4 Stroke** - This the most common engine type.  An engine cycle lasts two engine revolutions.  

**2 Stroke** - Less common engine type.  An Engine cycle lasts one engine revolution.  

Rotary engines are regarded as 2 stroke engines
