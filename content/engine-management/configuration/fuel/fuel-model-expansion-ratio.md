## Fuel Model: Expansion Ratio

![Image](</img/Config Fuel27.jpg>)

0: OFF

1: ON - EMAP1  Sensor

2: ON - EMAP Estimation Table

3: ON - EMAP Sensor 1/2  AVG

This uses the  of EMAP / MAP combined with the Static Compression Ratio to correct the VE of the  engine at different loads.

EMAP can be sourced from:

1) Actual EMAP from the Exhaust Manifold Pressure Input Channel

2) Estimated EMAP from the EMAP Estimation Table

  (Tuning view -> Fuel -> Compensations)
