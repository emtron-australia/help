---
title: "Custom TX Data Sets"
weight: 103
---

Emtron can be configured to send Custom Messages over CAN bus channels. 

To send the Custom TX Data sets, select them under 

Config -> Communications -> CAN Bus 1/2 -> DATA Set

** Runtimes selected in each data set conform to pre-defined scaling.  See Parameter Scaling and Parameter Scaling 2 under Additional CAN Information

** Runtime Channel Count per Data Set must be considered.  See CAN 1 / CAN 2 under Additional CAN Information
