## VVT Input Channel Selection

For closed loop position control, each camshaft must be assigned a position sensor.  \

Config View -> Inputs -> Input Pin Setup -> VVT 

Example Config (quad cam VVT control): 

![Image](</img/NewItem107.png>)

Configure sensor inputs as required.  Magnetic, Hall, DI threshold table, etc.  

******* Since one Cam position sensor is being used for sync, select which camshaft should reference the Sync Sensor for position – IE Intake (LH). ***
