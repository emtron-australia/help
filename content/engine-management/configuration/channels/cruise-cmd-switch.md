**Notes on "Cruise Control Switch Type = Custom "**

When the Voltage setting = 0.0V the corresponding setting is disabled within the Cruise Command Switch channel. In the below example the "Enable Sw" setting is disabled. 

![Image](</img/Untitled246.png>)

As the Enable Sw is disabled within the "Cruise Command Switch" channel in the above example, it allows the dedicated  "Cruise Enable Switch" channel to be used.
