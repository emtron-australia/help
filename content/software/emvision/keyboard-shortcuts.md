Shortcuts are also listed in **Help → Keyboard Shortcuts** within the application.

## File and Panels

| Key | Action |
|-----|--------|
| F6 | Show math channel setup |
| F7 | Toggle file management drawer |
| F8 | Toggle measurements drawer |

## Analysis Mode

| Key | Action |
|-----|--------|
| F9 | Toggle time / distance X-axis mode |
| F10 | Toggle track map follow mode (track vs car) |
| D | Toggle dual cursor mode |
| S | Swap primary and secondary cursor |
| B | Move secondary cursor to primary |

## Page and Pane Navigation

| Key | Action |
|-----|--------|
| Page Up | Select previous page tab |
| Page Down | Select next page tab |
| Ctrl+Page Up | Select previous plot pane |
| Ctrl+Page Down | Select next plot pane |
| Ctrl+P | Show page search |
| Ctrl+W | Show workspace search |
| Ctrl+Q | Close selected page |
| Ctrl+Shift+Q | Close all pages |
| Ctrl+1 | Set workspace layout to 1 pane |
| Ctrl+2 | Set workspace layout to 2 panes |
| Ctrl+3 | Set workspace layout to 3 panes |
| Ctrl+4 | Set workspace layout to 4 panes |
| Ctrl+` | Toggle auxiliary page |

## Log Comparison

| Key | Action |
|-----|--------|
| [ | Select previous active race section |
| ] | Select next active race section |
| ; | Toggle visibility of current section |
| , | Reduce offset of selected section |
| . | Increase offset of selected section |
| Ctrl+, | Quickly reduce offset |
| Ctrl+. | Quickly increase offset |
| ' | Set offset to cursor (or adjust by cursor difference in dual mode) |

## Plot Navigation

Arrow keys pan and zoom the plot view. Hold a key for accelerated scrolling. Hold **Ctrl** for faster movement (speed configured in Settings → Navigation).
