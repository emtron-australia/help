The **Live Data** drawer on the right side of the window shows all channel values and events from the telemetry stream.

## Channel Values

The upper section lists every channel currently being received, with its live value and units. Values update continuously while connected.

Use this list to:

- Verify that expected channels are being transmitted
- Identify channels to add to grids or plots
- Monitor values that are not assigned to a visible gauge or plot

## Events

The lower section shows recent system events as they arrive — engine start, lap beacons, alarm triggers, and similar discrete signals.

## Channel Preferences

Channel display units and decimal places are configured in the Live Data setup. Open the channels configuration to adjust how values are formatted.

Preferences apply to EmStream only and do not affect the display or log files. This allows team members in different regions to view the same telemetry stream in their preferred units.

## Drawer Size

Drag the drawer edge to resize. The drawer can be collapsed entirely — the width and open/closed state are saved between sessions.
