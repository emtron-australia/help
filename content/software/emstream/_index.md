EmStream is the PC client for receiving live telemetry data from Emtron displays. Connect to a vehicle, monitor channels in real time, and communicate via voice and text.

Start with the [EmStream User Guide](emstream) for an overview, then see [Getting Started](getting-started) for first-time connection.
