Accurate engine torque modelling is critical for transmission performance.

## Inertia

| Engine / configuration             | Typical Inertia (kg·m²) |
| ---------------------------------- | ----------------------- |
| Small 4-cyl light flywheel         | **0.12 – 0.20**         |
| Typical 4-cyl production           | **0.18 – 0.28**         |
| Performance 6-cyl (light flywheel) | **0.20 – 0.32**         |
| Heavier 6-cyl / street flywheel    | **0.28 – 0.40**         |
| Big heavy flywheel / truck         | **0.40 – 0.70+**        |
